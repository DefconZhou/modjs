
## <a href="#help" name="help">help</a>

Usage: `mod -h` or `mod --help` or `mod help`

Example:

```shell

mod help

# output
                          __   _
   ____ ___   ____   ____/ /  (_)_____
  / __ `__ \ / __ \ / __  /  / // ___/
 / / / / / // /_/ // /_/ /  / /(__  )
/_/ /_/ /_/ \____/ \__,_/__/ //____/
                        /___/

Usage: mod COMMAND [ARGS]

Commands:
  help           Get help on mod
  search         Search for module
  install        Install a module
  update         Update module version
  uninstall      Uninstall a module
  clean          Removes previously generated files and directories
  ls             List installed modules
  clear-cache    Clear local module cache
  publish        Publish a module
  unpublish      Remove a module from the repository
  link           Symlink a module folder
  rebuild        Rebuild project
  min            Minify js css html image files
  lint           Validate js css files
  compile        Compile
  create         Generate a project skeleton include project directory
  init           Generate a project skeleton in target directory
  server         Start a static web server
  pack           Create a tarball from a module
  hash           Rename file with it hash value
  replace        Replace the contents of files
  cat            Concatenate the content of files
  cp             Copy one or more files to another location
  mkdir          Create new folder
  mv             Move or rename files or directories
  rm             Remove files
  strip          Source stripping
```
